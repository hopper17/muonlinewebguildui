Política de Privacidad

Bot: REV3NGE BOT · Responsable: Hopper · Vigente desde: 23/08/2025

1. Resumen

REV3NGE BOT procesa datos para operar funciones en servidores de Discord, generar estadísticas de PvP y responder comandos. No vendemos datos.

Este documento es un modelo informativo y no constituye asesoría legal.

2. Qué datos recopilamos

  • Datos de Discord: IDs de usuario/servidor/canal, alias, mensajes y adjuntos usados para los comandos.
  • Archivos de log: ficheros ddmmyy.txt subidos por la comunidad a Google Drive con eventos de PvP.
  • Listas de facciones: datos leídos de Google Sheets API para clasificar jugadores.
  • Metadatos técnicos: fecha/hora de ejecución, errores y métricas básicas del bot.

3. Para qué usamos los datos

  • Responder comandos (!topkill, --name, etc.) y generar rankings.
  • Clasificar personajes por bando/rol.
  • Mantenimiento y seguridad (logs de errores, prevención de abuso).

4. Dónde y con quién compartimos

  • Discord (plataforma donde opera el bot).
  • Google Drive (almacenamiento de logs).
  • Google Sheets API (clasificación de jugadores).

No vendemos datos. Compartimos solo lo necesario para operar el servicio.

5. Retención

Los datos se conservan el tiempo necesario para el funcionamiento del bot y las estadísticas.

6. Seguridad

Aplicamos medidas razonables (control de acceso, tokens ocultos, mínimos privilegios).

7. Tus derechos

8. Menores

El bot está dirigido a mayores de 13/16 años (según país).

Publicaremos actualizaciones de esta política con nueva fecha de vigencia.
